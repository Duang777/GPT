FAQ

Questions before install

Does GPT Voyager send my chats to external servers?

No. Current implementation is local-first and stores data in browser local storage by default.

Which browsers are supported?

Chromium-based browsers (Chrome and Edge) with Manifest V3 support.

Can I migrate data between devices?

Yes. Export JSON backup from one device and import it on another device.

How do I load the extension manually?

Build project, open chrome://extensions, enable developer mode, then load dist/.
